# 网上音乐播放系统

## 登录注册模块

实现用户登录和注册功能

## 个性推荐

根据用户的音乐喜好推荐多种类型的歌单

## 歌单列表

根据音乐分类，展示多种歌单，用户可点击分类进行筛选

用户进入歌单列表，可点击播放全部音乐，或点击歌单列表中的列表项播放歌曲

## 播放详情界面

展示音乐图片，歌词，调节播放声音等

 

主界面

个性推荐：推荐歌单+独家放送+最新音乐

歌单：分类查询所有歌单（有播放界面）

mv:最新mv + 推荐mv（点击可展示MV详情)

最新音乐 ：根据分类展示歌单，点击可播放
